# Replay Engine

The replay engine reconstructs portfolio state for every trading day.

## Required data
- complete transaction history
- external capital events
- daily adjusted prices for every traded symbol
- daily benchmark prices

## Comparison portfolios
- actual LadderIQ portfolio
- same-selection buy-and-hold portfolio
- QQQ benchmark using matched cash flows

## Outputs
- equity curves
- returns by selected date
- alpha vs. QQQ
- ladder added value
- value added in dollars
- drawdown and volatility metrics
