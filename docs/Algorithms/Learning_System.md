# Learning System

The learning system converts repeated overrides into testable recommendations.

## Example
System recommends 3 NVDA shares at $214. User sells 2 at $210.085 to create a micro-harvest while retaining higher layers.

This becomes:
- an override event
- a stated reason
- an outcome to evaluate
- possible evidence for a future micro-harvest rule

No strategy rule changes until evidence meets the configured threshold and the user approves.
