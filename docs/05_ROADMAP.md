# LadderIQ Roadmap

## 3.53.x — Documentation and control baseline
- Semantic versioning.
- Central version metadata.
- Full documentation structure.
- Release checklist and changelog discipline.
- Centralized dashboard template.

## 3.54.x — Audited performance engine
- Capital-event ledger.
- Personal ROI.
- Time-weighted return.
- Money-weighted return / XIRR where appropriate.
- QQQ cash-flow-matched benchmark.
- Buy-and-hold replay.
- Ladder added value in percent and dollars.

## 3.55.x — Ladder QA engine
- Sell levels above market by validated spacing.
- Buy levels below market unless explicitly marked breakout.
- Share totals cannot exceed holdings.
- Cash allocations cannot exceed deployable capital.
- Position-limit warnings.
- Build failure on critical validation errors.

## 3.56.x — Learning foundations
- Structured override reasons.
- Outcome tracking.
- Strategy-profile proposals.
- Confidence thresholds.
- User approval and rollback.

## 4.0.0 — Adaptive product release
Potential scope:
- Operational Learning Engine.
- Investment DNA.
- Strategy-profile versioning.
- Multi-account portfolio model.
- Major UI redesign.
- Broker-read integration.

## Later horizons
- Retirement relocation analyzer.
- Tax-aware withdrawal engine.
- Mobile and cloud synchronization.
- Multi-user productization.
- Brokerage execution integrations, subject to compliance and safety controls.
